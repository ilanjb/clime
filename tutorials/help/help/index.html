Help can be added per attribute via the metadata kwarg of attrs.ib 

import attr

from clime import clime


@attr.s  # dont need auto_attribs, but it wouldn't hurt...
class Dude:
    name: str = attr.ib(
        metadata={"help": "Your first name"}
    )  # help per attibute is taken from the metadata kwarg in attr.ib

    def introduce(self):
        print(f"hi! my name is {self.name}")


def main():
    clime(Dude).introduce()


if __name__ == "__main__":
    main()
Check it:

python tutorial_001.py --help
usage: tutorial_001.py [-h] name

positional arguments:
  name        Your first name

optional arguments:
  -h, --help  show this help message and exit
